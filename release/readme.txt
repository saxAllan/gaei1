プログラムの使用方法


１．起動

    Python (3.6.2 以降) 環境で "main.py" を実行してください。
    ※このプログラムでは、以下のライブラリを利用しています。Anaconda 等を利用し、以下のライブラリが利用できる環境でご利用ください。
        statistics
        scipy
        matplotlib
        numpy


２．起動後の操作

    起動直後に読み込むデータファイルが尋ねられます。拡張子を除いた、ファイル名のみを入力し Enter を押してください。
    その後は VRML ファイル出力まで全て自動的に行われます。できるだけ処理内容を表示するようにしてありますので、進捗が分かりやすくなっています。


３．それぞれのファイルについて

    norminput.py : .dat ファイルを読み込み、プログラム内で扱いやすいようにデータを格納するプログラムです。同時に正規化（地図データの上下左右の非直線部分を取り除く作業）を行っています。
    mpointsl.py : データの欠損点を補間するプログラムです。
    judgements3.py : 地面判定処理を行うプログラムです。
    main.py : 上記のプログラムを正しい順序で動作させ、地図データを加工した後、VRMLデータを生成するプログラムです。
    final.wrl : このプログラムで後楽園周辺のマップをVRMLデータにしたサンプルです。
    multi_judge(failed).py : データの削減用に試作したプログラムです。設計が詰められず、今回のプログラムでは利用できておりません。
